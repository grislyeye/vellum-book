Vellum Book

A template project for publishing rich web documents using plain markdown. Very much a work-in-progress at this time, use with caution.

Demo | Roadmap

Features:

  • Write everything in plain markdown files
  • Rich document view with <vellum-doc>
  • Tooltip previews for local links with <link-preview>
  • Smartly converts links between markdown files into fragment links (e.g. Chapter 1)
  • Beautifully themed with magick.css by default

Design Principles

  1. Writers should only need write in plain markdown. Noodly HTML (like <link-preview> or <vellum-doc>) should be handled by 11ty.
